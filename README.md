# Ray Tracing Demo
A simple **2D CPU-based ray tracing demo** implemented in C++.
This project explores core ray tracing concepts such as ray casting, intersections, and basic lighting in a deliberately minimal 2D setting. The focus is on understanding rendering fundamentals rather than graphics API complexity.
